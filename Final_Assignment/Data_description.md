# Data Description

In the problem description there are 3 target points about the problem:

* Population Density.
* Venues in each city.
* Average income by state.

Population Density: It will be to estimate the city size, the information will be fetched from https://en.wikipedia.org/wiki/List_of_cities_in_Mexico, using the 2020 census. It will be necessary to parse the coordinates to the respective system.

Venues in each city: It will be fetched from Foursquare in a similar way to the third assignment of this course.

Average income by state: I consider it is the most important characteristic of the city for this problem. The information will be taken from https://infogram.com/sueldo-promedio-mensual-por-entidad-federativa-1g502yv3j6o62jd.
